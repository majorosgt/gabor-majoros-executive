# Dr. Gábor Majoros — Executive Self-Branding Website

## Project Overview
A professional executive branding / personal portfolio website for Dr. Gábor Majoros, highlighting his expertise in AI governance, privacy, and digital trust.

## Tech Stack
- **Language**: HTML5, CSS3, vanilla JavaScript (minimal)
- **Build system**: None (static site, no build step required)
- **Package manager**: None

## Project Structure
```
index.html   # Single-page website (all HTML, CSS, and JS inline)
replit.md    # This file
```

## Running the Project
The site is served by Python's built-in HTTP server on port 5000:
```
python3 -m http.server 5000 --bind 0.0.0.0
```
The workflow "Start application" handles this automatically.

## Deployment
Configured as a **static** deployment — Replit serves `index.html` directly from the project root.
